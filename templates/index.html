Type 86 Shipment Demo

Welcome to the Type 86 Shipment Processing Demo! This prototype demonstrates the process of onboarding a customer and submitting a shipment under Type 86 regulations with a fallback to Type 11. Follow the steps below to explore the functionality:

Demo Steps

  1. Onboard a Customer: Register a new customer to generate an API token for shipment submission. Start Onboarding
  2. Submit a Shipment: Use the API token to submit a shipment for processing. Submit a Shipment
  3. Explore the API: View the API documentation and test the endpoints using Swagger UI. View API Documentation

Note: The forms are prepopulated with dummy data for easy testing. Customer and shipment data are stored in data/customers.json and data/shipments.json, respectively.
